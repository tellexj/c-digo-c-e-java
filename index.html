O que são?


As linguagens de alto nível são geralmente mais portáveis, permitindo que o código seja executado em diferentes sistemas operacionais com poucas alterações ( C++ e Java ). Já as linguagens de baixo nível estão mais próximas do hardware e podem exigir ajustes específicos para cada plataforma, tornando o código menos portável (Assembly).
